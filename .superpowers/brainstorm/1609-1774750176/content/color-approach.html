How should we introduce more colour?

Your portfolio already defines cyan, purple, green, orange & pink — they're just not all visible yet. Pick the approach that feels right.

A

Accent text highlights

Use the existing palette more on headings, labels, and section numbers. Each section gets a distinct colour accent — e.g. purple for Skills, orange for Projects, pink for Certificates.

01 / Hero — cyan 03 / Skills — purple 04 / Projects — orange 05 / Certs — pink 06 / Contact — green
B

Coloured tags & chips

Skill tags, certificate badges, and project tech labels each get a coloured border or tint — pulling from the existing palette. The structural chrome stays dark; colour lives in the detail layer.

C

Both A + B combined

Section numbers and divider lines get per-section accent colours and tags/chips get coloured borders. More visual richness overall, still cohesive since all colours come from the existing palette.
